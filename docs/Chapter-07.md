### Chapter 07: Working with files

- AWS S3 Bucket

<br/>

    $ aws s3 mb s3://aunt-marias-pizzeria --region ${AWS_DEFAULT_REGION}

    $ cd api/pizza-image-processor

    $ claudia create \
        --region ${AWS_DEFAULT_REGION} \
        --handler index.handler

        --bucket aunt-marias-pizzeria \
